# Contributing

Submit candidate surfaces, not generic edges.

This project maps bounded legitimate surfaces. A useful contribution does not say "X is connected to Y." It names the surface that makes the connection auditable.

## Accepted contribution format

Copy `contributions/templates/candidate-surface.md` into `contributions/inbox/` and fill it out.

## Examples of valid surfaces

- named advisory board
- official appointment list
- Companies House filing
- funding round
- public event roster
- policy report authorship
- procurement award
- customer deployment
- taskforce roster
- cohort or directory

## Not enough

- same government department
- same industry
- same office
- same broad institution
- same vibe
- unsourced personal proximity

## Maintainer control

Candidate submissions do not enter the hop graph. The maintainer screens them and decides what belongs in the source ledgers.
