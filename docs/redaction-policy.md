# Redaction Policy

This project is public-role-only.

## Excluded material

Do not import or publish private contact data, addresses, intimate disclosures, dietary notes, private attendee metadata, raw leaked fields, authentication tokens, or similar non-public personal data.

## Listing caveat

A directory listing means only that a name appeared in a public or reported list. It does not prove attendance, membership, endorsement, agreement, knowledge, guilt, or wrongdoing.

## Validation

The validator rejects known private/sensitive field names and requires every edge to carry source IDs, an evidence class, and a status.
